# 🎓 EduGenie – AI-Powered Learning Assistant

## 📌 Overview

EduGenie is an AI-powered educational assistant developed using Google Gemini API and FastAPI to make learning smarter, faster, and more interactive.

✨ Key Features:
- ❓ AI Question Answering
- 📖 Concept Explanation
- 📝 Quiz Generation
- 📄 Smart PDF Summarization
- 🛣️ Personalized Learning Paths
- 🎨 Simple and User-Friendly Interface

### 🛠️ Technologies Used
- Python
- FastAPI
- Google Gemini API
- HTML
- CSS
- JavaScript

### 🎯 Goal
To provide students with an intelligent learning companion that helps them understand concepts, generate quizzes, summarize study materials, and create personalized learning roadmaps.
